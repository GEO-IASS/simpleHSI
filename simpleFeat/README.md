##### Linear Methods

* Partial Least Squares
* Canonical Components Analysis

##### Nonlinear Methods

* Kernel Principal Components Analysis

* Kernel OPLS
* Kernel Canonical Components Analysis
* KMNF
* KECA
* Laplacian Score
